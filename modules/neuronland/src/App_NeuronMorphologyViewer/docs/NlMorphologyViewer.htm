    Overview

“NLMorphologyViewer” is a simple interactive 3D viewer for neuron morphology data files. It is built on top of the file format conversion technology developed for the “NLMorphologyConverter”, and supports the same 20 file formats which the original command line application supports, including Neurolucida, SWC, MorphML, NeuroZoom, Eutectics, and many others.

“NLMorphologyViewer” is designed to be a easier to use than “NLMorphologyConverter”, adding extra display functionality that allows a visual inspection of morphology data files.

The application is free, and (eventually) will be multi-platform. It is currently only available as a binary download for Microsoft Windows. There is a plan to release Apple Mac OSX and Linux versions in the near future.



    Main Features

  • Import 20 different file formats, with auto-detection of file format.

  • Neuron morphology 3D visualisation.

  • Neuron statistics and fault data window.

  • Manipulation of imported neuron data for visualisation and export.



    Installation

Windows

The Win32 version comes in a standard microsoft installer package (msi). Simply double-click on the installation package to install.





    Tutorial

      MENUS

        File

Import

This menu contains commands that are applied to the current neuron file, import a new file or close the application.

Import neuron data from the selected file. If the data is in a supported format, the file format will be automatically determined. Note that for large data files, and depending on the data format, the data may take a several seconds to load.

This command is always available to the user. If a neuron is imported while a previous neuron is still loaded, the previous neuron will be discarded automatically.

Export

Export current neuron data to a file.

      • Export Original Neuron As

    Saves the Neuron original data with the user selected name and format.

      • Export Visible Neuron As

    Saves the currently visible neuron data with the user selected name and format. (i.e. the neuron components which the user has not explicitly deactivated).



In both cases, the export dialog will only allow a neuron to be exported to a file format which supports the data. For example, trees with trinary branch points cannot be exported to formats which only support binary branch points.

The user has to select the location, name and format for the exported file data.

The commands are only activated while a neuron is currently loaded

Save Image

Not applicable in current version.

Close Neuron

    Close current neuron without exiting the application. The 3D view window will be cleared, and commands which require the presence of neuron data will be disabled.

This command is only activated while a neuron is currently loaded

Exit

    Closes the application.

        View

Neuron Components Data

Shows or hides the neuron components data panel on the left hand side of the window.

Sample Data Points & Sample Line Segments

Select the style of 3D visualisation of the Neuron. At the moment, two simple views are available. One shows all the original data points. The other displays the shape and connectivity of the data, by connecting sample points with lines.

Future versions will contain more sophisticated visualisations.

      PANELS

        Neuron Component Data

When active (it is active by default), this panel always resides on the left, and consists of two sub-panels..

Top Panel – Neuron Components List

Shows the list of all the neuron components (somas, dendritic trees and axon trees). All components may be selected and their colour or visibility modified. For each component the following information appears in the list:

  • Type of component (Soma, Dendritic Tree, Axon Tree)

  • Visibility - (Y)es or (N)o

  • Markers - not active in this version.

  • Color of the component in the 3D neuron window (the colour of the text matches the colour of the component in the 3D display).

To select a component mouse click on it. To select more than one, keep the “ctrl” key pressed while selecting.

The buttons for modifying the components are only enabled when there is one or more components selected in the list.

        • Button ON/OFF

Toggle the visibility of the selected neuron components. Non-visible components will be greyed-out in the list and will be not shown in the 3D neuron display.

        • Button Colour

Displays a Colour selection dialog to choose the colour of the selected components. Selecting a colour and pressing the accept button results in the components colour change to the new one.

Bottom Panel – Statistics and Warnings

Warnings Tab

Displays a list of warnings (if any) indicating unusual/unrealistic/problematic data related to the Neuron Data.

The following warnings might be shown

    Tree root too distant from soma

    Unary branch point

    Non-binary branch point

    Disjoint branch connection

    Branch has zero samples

    Branch formed from only one sample

    Sample diameter is excessively large

    Sample diameter is unusually small

    Sample diameter is zero

Unusually large change in diameter

    Large terminal diameter

    Unusually long section

    Unusually short section

    Intersecting sections

    Encapsulated branch (a child branch lies completely within the extent of its parent).

Statistics Tab

Displays the statistics for the neuron as a whole (if no individual component is selected in the list), or for the individual axonal/dendritic trees selected.

If there is one dendrite or axon component selected in the components list, it displays the component statistics, otherwise it displays the entire neuron ones.

The following Statistics data are shown:

    Number of sample points

    Number of branch points

    Branching order

    Number of terminals

    Total tree branch length (microns)

    Tree surface area (microns^2)

    Tree volume (microns^3)

    Maximum terminal path distance (microns)

    Maximum terminal physical distance (microns)



Neuron 3D Display



When a neuron is imported, it will be immediately displayed in the 3D window. Initially the neuron is rotating around the y axis, and the camera is positioned at a distance so that most of the neuron should be visible.



To stop the neuron rotating – left click in the window without moving the mouse



To take control of the neuron – keep the left mouse button down and drag the mouse in the direction you with the neuron to rotate.



To spin the neuron in a specific direction – left click the mouse button, keep it pressed, drag, and release...



Using a combination of these actions, the neuron can be manipulated (around the center of rotation, which cannot currently be altered) to any position in 3D space.
